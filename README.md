KityMinder Core For Case
==========

## 简介

KityMinder 是一款强大的脑图可视化/编辑工具，由百度 FEX 团队开发并维护。

本仓库基于KityMinder，专门为测试用例定制部分功能：

* 新增启用、禁用功能
* 新增节点类型（用例三要素：模块、用例、用例步骤）
* 新增用例执行状态（成功、失败、阻塞）

## 使用
